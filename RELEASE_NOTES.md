
# Quiver → Fletchy Converter v1.0.0

First public release of the **standalone browser converter** to help users migrate from **Quiver** to **Fletchy** — the modern Quiver app alternative.

## Highlights
- Client‑side conversion of `.qvlibrary` / `.zip` to `.fletchy`
- Drag‑and‑drop UI with progress stages and success state
- Preserves Markdown, code (with language), LaTeX, and basic structure
- No server, no data leaves your device

## Assets
- `Source code (zip)` and `Source code (tar.gz)` from GitHub
- Attach `index.html` directly for quick download if desired
